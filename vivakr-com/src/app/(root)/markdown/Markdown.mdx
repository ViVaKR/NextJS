# 쿠키 정책

ViVaKR.COM(이하 '서비스')은(는) 이용자에게 더 나은 서비스 경험을 제공하기 위해 쿠키(Cookies)를 사용합니다. 본 쿠키 정책은 쿠키의 사용 목적, 종류, 관리 방법 등을 안내하며, 이용자가 쿠키 사용에 대해 충분히 이해하고 선택할 수 있도록 돕기 위해 작성되었습니다.

## 1. 쿠키란?

쿠키는 이용자가 웹사이트를 방문할 때 브라우저에 저장되는 작은 텍스트 파일입니다. 이를 통해 웹사이트는 이용자의 기기를 인식하고, 맞춤형 서비스를 제공할 수 있습니다.

## 2. 쿠키 사용 목적

서비스는 다음과 같은 목적으로 쿠키를 사용합니다:

-   **필수 쿠키**: 서비스의 기본 기능(예: 로그인 유지)을 제공
-   **분석 쿠키**: 이용 패턴 분석을 통해 서비스 개선
-   **기능 쿠키**: 이용자의 설정(예: 언어 선택)을 기억

현재 광고 목적의 쿠키는 사용하지 않습니다.

## 3. 사용되는 쿠키의 종류

-   **세션 쿠키**: 브라우저 종료 시 삭제되는 임시 쿠키
-   **영속 쿠키**: 일정 기간 동안 저장되어 이용자 편의를 제공
-   **제3자 쿠키**: 현재 사용하지 않음 (사용 시 별도 공지)

## 4. 쿠키의 보유 및 이용 기간

쿠키는 목적에 따라 다르게 저장되며, 최대 1년간 보관 후 자동 삭제됩니다. 세션 쿠키는 브라우저 종료 시 즉시 삭제됩니다.

## 5. 쿠키 관리 방법

이용자는 쿠키 사용을 자유롭게 선택할 수 있습니다:

-   브라우저 설정에서 쿠키를 허용, 차단, 또는 삭제 가능
-   예: Chrome - 설정 - 개인정보 및 보안 - 쿠키 및 기타 사이트 데이터

단, 필수 쿠키를 차단할 경우 서비스 일부가 제한될 수 있습니다.
